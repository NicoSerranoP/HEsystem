2. Payment Connection Tutorial

A system without intermediaries or third-parties works thanks to the collaboration of thousands of computers around the world that mantain a decentralized network running all the time. To interact with this network without having to install and setup a specialized software, you can use the services of Infura. They would lend you a free connection through their own computers. This tutorial shows you how to connect to Infura to enable payment processes for our service.

1. Create an Infura account

Visit the Infura official website and create a free account. If you are developing more than three projects in the Ethereum network, then you would have to pay a monthly fee. 

2. Create a new project

In your Infura account, create a new project by clicking Ethereum/ Create new project.

3. Get your endpoint URL

You can find your HTTPS endpoint URL in the configuration tab of your new project. Remember to select the right network (Mainnet or Ropsten, check out the 6th step of this tutorial if you need help).  The endpoint URL is similar to:
https://mainnet.infura.io/v3/653dabde81094ba1a2c283d8a0c98af2
