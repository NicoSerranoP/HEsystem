Do you need private data from our organization to get useful information for you? With our system you would be able to perform any operation over hidden data (encrypted) and obtain a true and visible result for you. That is how you can take the maximum advantage of our database for your activities without compromising our intelectual property.

Accesibility

This system allows you to have access to private and/or sensitive data from an organization with great efficiency.

Privacy

The organization data and your own computations remain private all the time. The data is encrypted and all the computations are performed in your computer.

Trust

You can trust the system would return you the real result only for you and the organization can trust that they would receive their payment. All these without intermediaries or third-parties involved.

Tutorial

1. Create your Ethereum wallet

The Ethereum wallets are your credentials to purchase things in the Internet without the need of an intermediary by using the cryptocurrency Ether. If you need help creating it, check out this tutorial.

2. Set up your payment connection

The intermediary-less platforms or decentralized platforms run over a distributed computers network around the world. To connect with this platform, you have two choices: install a Ethereum client in your computer and configure it properly (it is complicated but you can check this tutorial for help) or you can create an account in a service which would lend you a remote client. Check this tutorial to connect to the service. 

3. Download our library

To buy and interact with the organization data in your own computer, you would need to use a specialized library and perform your computations in one of the most used programming languages for data science: Python. Check this  tutorial to download and configure the library.

4. Test your algorithm

Test your computations or algorithm that would interact with the organization data. Check this tutorial to setup the testing process. 

5. Buy access and get your result

After you have tested and confirmed your algorithm is working with the organization data, you can purchase access to the real data and process it with your algorithm. When you finish, the system will show you the final result. Check this tutorial to configure the purchase process.
