3. Library Tutorial

To interact with the organizations' encrypted data, you would have to install a specialized library in Python. The data would be encrypted using a homomorphic encryption protocol (Paillier protocol) and with out library you would be able to interact with the encrypted data: perform primitive operations like addition, multiplication, divisions, subtraction and comparisons). We have chosen the Python programming language to develop the library because it is one of the most used languages in data science and analytics.

1. Install Python

Install the easy-to-learn and most used programming language in data science to interact and process the data received from our organization. (Download the language here and if you need take a crash course of data science and machine learning here).

2. Download our library

The required functions to ask for encrypted data, process it and ask for the result have been previously implemented in our library. To use them download this repository, decompress it and put the hesystem folder in your working directory. 

3. Install all dependencies

In the downloaded repository you woul find a dependencies file (requeriments.txt) that you would have to install to guarantee the correct functioning of the library. To install it, open a terminal in your working directory and execute the following command: 

pip install -r requirements.txt
