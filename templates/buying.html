5. Purchase Tutorial

At this point you have checkout the previous tutorial and you have your algorithm or computations ready. Remember that you can ask for the encrypted data multiple times but you can only ask once for the final result. To purchase access, follow these steps:

1. Import the library

To handle the organization's data you would need to interact with your library functions (to install it check this tutorial). To import the library you can execute the following command in a terminal or inside a Python .py file:

import hesystem.client as cl

2. Set up your information

In this step you would need: your account address, private key (you can follow this tutorial to get them) and your URL endpoint (you can follow this tutorial to get it). Remember to have enough funds in your account to execute the purchase. In the same terminal or file from the previous step, execute the following command to enter your information: 

User = cl.initialize()

3. Request the data

Get the package details URL that you wish you access (you can find it here). In the same terminal or file from the previous step, execute the following command to enter the URL and request the data: 

data, public_key = cl.request_data(User)

4. Execute your algorithm

The variable data from the previous step contains the encrypted data from the organization. You can interact with it in different ways. If you need specific help to do this, visit the library documentation here.

result = data.transpose().mean(1)[0]

5. Request the result

When you algorithm or compuations have finished, you would have an encrypted result. Check out the conditions that your result should follow here. To request the decrypted result execute the following command in the same terminal or file from the previous step:

final_result = cl.request_result(User, result, public_key)

You have completed the process, you have the result you need without compromising the intelectual property of the organization.
